FAQ
Common Questions

Common
Questions

Everything you need to know before acquiring an IS Licensing Core asset. Not answered here? Reach us directly via the Contact page.

What does "single-entity" license mean?+
A single-entity license means the IS asset is licensed to one organization or individual. It cannot be resold, transferred, sublicensed, or shared across multiple entities. The license is bound to the organization named at the point of authorization.
How is the API key delivered?+
API keys are provisioned same-day upon successful authorization. They are delivered via the encrypted channel associated with your billing email. You will receive the key, endpoint documentation, and basic integration instructions within hours of payment confirmation.
Can I renew the API license after 30 days?+
Yes. The Neural Logic Front Desk API license is renewable on a 30-day cycle. You will receive a renewal notification 5 days before expiry. Renewal maintains the same API key and endpoint — no re-integration required.
What's included in the Systems Blueprint consultation?+
The Blueprint license includes a 60-minute IS scoping consultation with a professional, a full IS architectural design delivered as a detailed system map (PDF + source files), and one revision round. Additional revisions or expanded scope can be arranged separately.
Are these assets produced by AI?+
No. All IS assets are authored by human professionals with degrees in Business Administration and Information Systems. We don't sell AI-generated content. The value of our IS assets lies in the professional judgment, methodology, and domain expertise behind them.
Is there a refund policy?+
Due to the proprietary nature of the IS assets, licenses are non-refundable once provisioned. For API licenses, if there is a verified technical failure on our end preventing access within the first 48 hours, we will extend the license period accordingly.
Can I use the research document in client-facing work?+
The Academic Strategy License covers your own professional use, including using the document as a reference framework in your own work. It does not permit reproduction, redistribution, or direct resale of the document. For client-facing use cases, contact us about an extended licensing arrangement.
How long does Blueprint delivery take?+
The Blueprint process begins with a scoping consultation within 48 hours of authorization. Full IS architectural design delivery typically takes 7–14 business days depending on scope complexity. You will receive a timeline estimate after the scoping session.
What IS stack does the API integrate with?+
The Neural Logic Front Desk API is REST-based and JSON-native, making it compatible with any modern stack. It has been tested with major CRM platforms, custom Node.js and Python environments, and standard webhook-based automation pipelines.
